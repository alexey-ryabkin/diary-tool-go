# DiaryToolGo

Небольшая CLI-утилита на Go, которая создаёт текстовые файлы для одного месяца моего дневника.

Я написал её для себя — ради удобства и чтобы попрактиковаться в Go.

## Что делает

Программа:

- работает в текущем каталоге
- создаёт папку месяца (`YY.MM, Месяц`)
- создаёт внутри текстовые файлы для каждого дня месяца

Пример структуры:

```
26.03, Март/
2026.03.01, воскресенье.txt
2026.03.02, понедельник.txt
2026.03.03, вторник.txt
...
```

Уже существующие файлы **никогда не перезаписываются**.

## Использование

Скачать, установить, использовать:

```
git clone https://github.com/alexey-ryabkin/diary-tool-go.git
cd diary-tool-go
go install ./cmd/diary-tool-go
mkdir ~/Дневник
cd ~/Дневник
diary-tool-go
```

При запуске программа:

- определяет, какой последний месяц создан полноценно,
- предлагает создать текущий месяц,
- или следующий месяц (по умолчанию).

После выполнения выводится:

- сколько файлов создано,
- какие уже существовали,
- какие лишние файлы найдены в папке.
